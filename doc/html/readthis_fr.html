JoclyMatch

Un petit serveur de jeu pour jouer aux jeux Jocly, sans base de données ni suivi.

Ce site est basé à l'origine sur une expérimentation de Jérôme Choain, alias @jcfrog, permettant aux utilisateurs de jouer aux jeux Jocly entre eux afin de pallier l'arrêt du serveur jocly.com.

La version d'origine a été complétée et les sources sont disponibles sur GitHub :

https://github.com/fhoudebert/joclymatch

Jocly est une bibliothèque open source de jeux de plateau créée par Michel Gutierrez. Plus d'une centaine de jeux sont disponibles.

Les développements ont été relancés avec des contributions de HG Muller et François Houdebert. Les sources sont disponibles ici :

https://github.com/fhoudebert/jocly2

Avec joclymatch, vous devriez pouvoir installer ce petit serveur de jeu n'importe où.

À vous de décider ce qui constitue un avantage ou un inconvénient :

  • pas de base de données : il suffit d'un serveur web avec PHP. Les parties sont stockées dans de simples fichiers texte lisibles (JSON) ;
  • pas de connexion, pas de mot de passe (cela pourra évoluer). Aucune donnée personnelle n'est stockée, uniquement les coups joués ;
  • pas de publicité, pas de suivi ;
  • une interface très basique permettant simplement de :
    • jouer seul contre l'ordinateur ;
    • créer une partie : vous devrez ensuite envoyer un lien à votre adversaire ;
    • fonctionner avec un système de consultation périodique classique (polling).

Vous pouvez tricher : toute personne disposant du bon lien peut jouer du côté A ou du côté B.

Mais nous sommes des gentlemen, n'est-ce pas ?

Il est également possible d'utiliser la version push à la place du polling, si vous disposez d'un VPS avec un serveur Node.js.
